Тестовое задание для компании [FLOYD](https://ftoyd.com/)

Запуск программы:
1. git clone https://github.com/DamarooWork/ftoyd-test.git
2. cd ftoyd-test
3. npm i
4. npm run dev

Технологии:
1. Next.js
2. TailwindCSS
3. Typescript
4. GSAP
5. reduxjs/toolkit
6. socket.io-client
7. react-loader-spinner

Deploy: https://ftoyd-test.vercel.app/
